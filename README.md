# 🌸 FlowerStore - Online Flower Shop

FlowerStore is a simple e-commerce website where users can browse and purchase flowers, bouquets, wedding decorations, and event arrangements. This project is built with **HTML, CSS, JavaScript, Bootstrap**, and optionally a **Spring Boot backend** for product management.

---

## 📌 Features

- Browse products with images, prices, and descriptions  
- Add products to cart  
- Update product quantity in cart  
- Remove products from cart  
- View total amount dynamically  
- Checkout functionality  
- Sign In simulation (localStorage-based)  
- Responsive design using Bootstrap  
- Footer with contact, social media, and location
